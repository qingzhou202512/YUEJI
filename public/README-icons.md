# PWA 图标说明

为了让 PWA 功能完整，需要添加以下图标文件：

## 需要的图标文件

1. **icon-192.png** (192x192 像素)
   - 用于主屏幕图标（Android）
   - 建议使用圆角设计，符合应用风格

2. **icon-512.png** (512x512 像素)
   - 用于安装时的显示（Android）
   - 建议使用圆角设计，符合应用风格

## 图标设计建议

- 使用应用主色调（紫色 #8B5CF6）
- 简洁的图标设计，易于识别
- 确保在小尺寸下也清晰可见
- 可以使用圆角矩形背景

## 临时方案

在开发阶段，可以使用在线工具生成占位图标：
- https://www.pwabuilder.com/imageGenerator
- 或者使用简单的纯色图标作为占位

## 添加图标后

将图标文件放在 `public/` 目录下，Vite 会自动复制到构建输出目录。
